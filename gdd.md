# Game design Document

吸血鬼在三天之内逃出诅咒之地——黑夜吞食敌人积攒精力，白天限时边逃亡边杀怪

## 阶段

每局游戏分为**白天**阶段和**黑夜**阶段

每局起始阶段为黑夜，之后白天与黑夜依次交替，共计 5 个阶段

（黑夜0 -> 白天1 -> 黑夜1 -> 白天2 -> 黑夜2）

阶段切换：

- 白天 -> 黑夜：生命值回满，精力归零
- 黑夜 -> 白天：生命值回满，精力保留

地图：

- 黑夜：正方形
- 白天：窄宽矩形

倒计时：

- 黑夜：与关卡相关的倒计时
- 白天：与角色精力相关的限时

## 关卡

每局游戏分为**人类**关卡和**怪物**关卡

每当阶段切换，玩家自选关卡

## 成长

吞食或击杀敌人概率掉落经验球，角色拾取经验球提升进度条

- 黑夜：只要进度条满时，就弹窗选卡（立即选卡）
- 白天：只有阶段结束时，才弹窗选卡（延迟选卡）

- 人类关卡：属性卡
- 怪物关卡：技能卡

## 操作

- `WASD` 移动（在白天阶段 −2 精力/second）：
- `Space` 闪避（白天阶段使用 −15 精力/eachtime）：
- `E` 吞食（白天阶段不可使用）：当敌人的生命值低于 10% 时，敌人进入逃跑状态，此时敌人可被角色吞食，角色 +10 精力/unit
