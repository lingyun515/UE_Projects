
# Bullet

Bullet 不是子弹，Bullet 是攻击判定载体

不管是敌人的 Bullet，还是玩家的武器 Bullet ，都用同一套 Bullet 系统。

- 初始位置：Bullet 从哪出现
- 形状（碰撞体）：点状、条状等等，判定是否为与实体碰撞
- 预警时间：Bullet 出现时提前多久预警（为 0 即代表其无预警出现）
- 存活时间：Bullet 存在多久后消失
- 移动方式
- 碰撞效果

Bullet 移动方式：

- 在初始位置一直静止
- 从初始位置朝某处直线发射
- 从初始位置追踪某物（实现上，或为每帧以直线朝某物直线发射）
- 从初始位置绕着某物做圆周运动
- ……

Bullet 撞到不同东西有不同效果：

- 撞到墙体：
	- 可以让自己消失（单体伤害）
	- 可以让自己反弹
- 撞到敌方实体：
	- 可以让自己消失（单体伤害）
	- 可以不让自己消失，按照原本的移动方式继续走（穿透伤害）
	- 无论哪种，都应对目标造成伤害
- 撞到友方实体：
	- 也许可以造成友伤

# 武器生成的 Bullet

所有武器的 Bullet 就是预警时间恒为 0 的 Bullet

对于近战武器（刀枪棍棒……）的 Bullet：

- 初始位置为武器使用时的当前位置
- 碰撞体与武器碰撞体等大
- 存活时间根据攻速决定
- 移动方式从初始位置画圆弧

对于远程武器（弓弩……）的 Bullet：

- 初始位置为武器使用时的当前位置
- 碰撞体与武器碰撞体等大
- 生成频率根据攻速决定
- 存活时间另外设定
- 移动方式从初始位置朝距离最近敌方实体直线平移

对于光环武器（法阵……）的 Bullet：

- 初始位置为武器所有者的当前位置
- 碰撞体为大型点状（圆）
- 存活时间根据攻速决定
- 移动方式为跟随武器所有者平移

对于轨道武器（法球……）的 Bullet：

- 初始位置为武器所有者的当前位置
- 碰撞体与武器碰撞体等大
- 存活时间另外设定
- 移动方式为跟随武器所有者平移+绕武器所有者旋转（运动的合成）

## 怪物 Bullet

对于近战型怪物 Bullet

- 可能的实现
	- 初始位置为该怪物与敌方实体碰撞的当前位置
	- 碰撞体与该怪物的碰撞体等大
	- 存活时间为 1 帧
	- 移动方式为不动
- 可能的实现
	- 初始位置为该怪物的初始位置
	- 碰撞体与该怪物的碰撞体等大
	- 撞到敌方实体不会消失
	- 存活时间为该怪物的存活时间
	- 移动方式为该怪物的移动方式

对于远程型怪物 Bullet

相当于该怪物拥有远程武器
